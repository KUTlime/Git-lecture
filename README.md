# Pokyny k odevzdání cvičení pro získání certifikátu

V repozitáři Git-lecture, na větvi „odevzdani“, odkaz zde: [KUTlime/Git-lecture at odevzdani (github.com)](https://github.com/KUTlime/Git-lecture/tree/odevzdani)
máte připravenou šablonu pro odevzdání, které po Vás budu chtít. Každá kapitola domácích úkolů má samostatný textový soubor, tzn. GitClone.txt, GitAdd.txt atd.

Vaše vypracování vepište do prázdných řádků. Při odevzdání mne zajímá pouze syntaxe příkazů gitu. Jak si vytvoříte složky, soubory a jak je upravíte je již na Vás. Příklad:

```txt
# Zadání 1
git add Soubor.txt
# Zadání 2
git add Soubor2.txt
```

Pakliže není v zadání řečeno jinak a zadání obsahuje více než jeden příkaz Gitu, napište sled příkazů pod sebe. Příklad:

```txt
# Zadání 1
git add Soubor.txt
git commit …
git status …
# Zadání 2
git add Soubor2.txt
```

Pokud zadání nebo záhlaví kapitoly příkladů obsahuje pokyn pracovat s veřejným repozitářem, přiložte odhad na připravené místo do příslušného souboru. Příklad:

```txt
# Odkaz na veřejný repozitář
https://github.com/KUTlime/Git-lecture
# Zadání 1
git add …
```

Na veřejném repozitáři, jehož odkaz uvedete, prosím aplikuje příkazy z dané kapitoly výsledek odešlete do vzdáleného repozitáře, aby šla vidět historie komitů a výsledek.

Jakmile budete mít hotové vypracování, soubory GitAdd.txt, GitClone.txt a ostatní uložte do nového veřejného repozitáře, určeného pouze pro potřeby odevzdání a pošlete mi na něj odkaz email, který máte již k dispozici.

Vyhodnocení proběhne přesně týden po skončení kurzu. Máte tak možnost se ještě vrátit k vypracování a upravit výsledky dle potřeby i po zaslání odkazu emailem.
